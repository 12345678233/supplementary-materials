# Anonymous Supplementary Material

This repository contains compact supplementary materials for the rebuttal.  
It is organized into five parts, each serving a different role in clarifying the paper's evaluation perspective.

## Contents

### Part 1. GSA-style evaluation worked example
A concrete worked example showing how the GSA framework can be operationalized in an embodied setting, with G/S/A distinguished by evidence type rather than benchmark difficulty.  
[Open Part 1](./part1_gsa_style_evaluation.md)

### Part 2. Interpretable task decomposition and difficulty analysis
An anonymized example showing how benchmark items can be decomposed into underlying ability requirements and analyzed by difficulty, to make evaluation more interpretable.  
[Open Part 2](./part2_task_decomposition_difficulty_analysis.md)

### Part 3. Original eight household tasks
The original fixed eight-task embodied household benchmark example, including task categories, system framework, and benchmark results.  
[Open Part 3](./part3_original_eight_household_tasks.md)

### Part 4. Autonomous task generation results
Quantitative results for the autonomous task generation framework, including task scale, diversity, integration rate, human validation, and spatial statistics.  
[Open Part 4](./part4_task_generation_results.md)

### Part 5. Autonomous task generation paradigm
A concise description of the closed-loop paradigm for autonomous task generation through exploration, environment-to-task translation, filtering/diversification, and task evolution.  
[Open Part 5](./part5_task_generation_paradigm.md)

## Suggested reading path

- For the overall evaluation logic: start with **Part 1**
- For interpretability of benchmark design: see **Part 2**
- For the original embodied benchmark example: see **Part 3**
- For evidence on autonomous task generation: see **Part 4** and **Part 5**
